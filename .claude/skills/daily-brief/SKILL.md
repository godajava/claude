# 아침 브리핑 생성

오늘 일정과 할 일을 요약해 아침 브리핑을 만듭니다.

## 절차

1. KST 기준 오늘 날짜를 확인합니다: `TZ=Asia/Seoul date +"%Y-%m-%d (%a)"`
2. `secretary/schedule.md`에서 오늘 날짜 섹션(및 내일 섹션)을 읽습니다.
3. `secretary/todo.md`에서 미완료(`- [ ]`) 항목을 모두 읽습니다.
4. 할 일을 다음 기준으로 분류합니다 (KST 기준):
   - **밀린 항목** 🔴 — 마감일이 오늘 이전
   - **마감 임박** 🟡 — 마감일이 오늘부터 3일 이내
   - **일반** — 그 외 (마감 없는 항목 포함)
5. 아래 형식으로 브리핑을 출력합니다.

## 출력 형식

```
☀️ YYYY-MM-DD (요일) 아침 브리핑

📅 오늘 일정
- HH:MM 제목 — 장소 (메모)
(없으면: "오늘 예정된 일정이 없습니다.")

✅ 할 일
🔴 밀림: 항목 (마감: YYYY-MM-DD, N일 지남)
🟡 임박: 항목 (마감: YYYY-MM-DD, D-N)
- 일반 항목

📌 내일 미리보기
- 내일 일정이 있으면 한 줄로 요약

한 줄 코멘트: 오늘 가장 중요한 것 하나를 짚어줍니다.
```

## 규칙

- 파일이 없거나 비어 있으면 해당 섹션에 "기록 없음"이라고 적고 넘어갑니다. 파일을 새로 만들지는 않습니다.
- 브리핑은 읽는 데 30초를 넘지 않도록 간결하게 작성합니다.
- 밀린 항목이 있으면 브리핑 맨 위에 개수를 강조합니다 (예: "⚠️ 밀린 할 일 2건").
