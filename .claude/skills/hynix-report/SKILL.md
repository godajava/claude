# SK하이닉스 투자 리포트 생성

SK하이닉스(KRX: 000660)의 주가·뉴스·산업 동향을 조사해 한국어 투자 리포트를 작성합니다.

## 절차

1. KST 기준 오늘 날짜를 확인합니다: `TZ=Asia/Seoul date +"%Y-%m-%d (%a)"`
2. **주가 데이터** — WebSearch(`"SK하이닉스" 주가 마감 YYYY-MM-DD`, "SK하이닉스 시가총액 PER")로 확인합니다: 현재가/종가, 전일 대비 등락률, 52주 최고·최저, 시가총액, PER·PBR. (참고: finance.naver.com은 WebFetch가 차단되므로 WebSearch를 사용합니다.)
3. **뉴스·공시** — WebSearch로 최근 1주일 내 주요 뉴스와 DART 공시를 조사합니다 (실적, 수주, 투자, HBM/DRAM/NAND 관련).
4. **산업 동향** — 메모리 반도체 업황을 조사합니다: DRAM/NAND 가격 추이, HBM 수요(AI 서버), 경쟁사(삼성전자, 마이크론) 동향.
5. **증권사 의견** — WebSearch("SK하이닉스 목표주가")로 최근 국내 증권사 리포트의 투자의견·목표주가를 수집합니다.
6. `reports/sk-hynix/`에 기존 리포트가 있으면 가장 최근 것을 읽어 직전 리포트 대비 변화를 파악합니다.
7. 아래 형식으로 리포트를 작성해 `reports/sk-hynix/YYYY-MM-DD.md`로 저장합니다.
8. 저장 경로와 3줄 이내 핵심 요약을 사용자에게 보고합니다.

## 2. 주요 뉴스·공시 (최근 1주)
- YYYY-MM-DD 제목 — 한 줄 요약 (출처)

## 3. 산업 동향
- DRAM/NAND 가격, HBM 수요, 경쟁사 동향을 3~5개 불릿으로 정리

## 4. 증권사 투자의견
- 증권사명: 투자의견 / 목표주가 ₩ (날짜)

## 5. 투자 포인트
- 긍정 요인 2~3개

## 6. 리스크 요인
- 부정 요인 2~3개

## 7. 종합 의견
직전 리포트 대비 변화를 포함해 3~5문장으로 정리. 사실과 해석을 구분해 서술.

## 규칙

- 확인하지 못한 수치는 지어내지 말고 "확인 불가"로 표기합니다.
- 주요 수치·인용에는 출처(언론사·증권사·DART)를 밝힙니다.
- 같은 날짜 리포트가 이미 있으면 덮어쓰지 말고 내용을 갱신(수정)하고, 갱신 사실을 보고합니다.
- 매수/매도를 직접 권유하는 표현은 쓰지 않습니다. 면책 문구는 반드시 포함합니다.
